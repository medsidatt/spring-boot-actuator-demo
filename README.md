---

## üß© What is Spring Boot Actuator?

**Spring Boot Actuator** is a module that adds several management and monitoring
features to your application with almost no configuration.  
Actuator exposes useful **HTTP endpoints** that reveal information about your application,
such as health status, metrics, logs, environment details, and more.

It is commonly used for:
- Health checks
- Metrics collection
- Application monitoring
- Debugging
- Integration with tools like Prometheus, Grafana, Kubernetes, and Spring Admin
